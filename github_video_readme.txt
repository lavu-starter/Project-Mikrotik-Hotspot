## Cara Menjalankan Proyek

1. Clone repositori:
   ```bash
   git clone https://github.com/username/repository.git
   ```
2. Buka proyek di editor favorit Anda.
3. Ikuti panduan di video untuk memahami langkah-langkah konfigurasi Hotspot Mikrotik.

## Keterangan
Proyek ini dibuat sebagai contoh untuk menampilkan cara menambahkan link video pembelajaran Hotspot Mikrotik dari Google Drive ke GitHub.
